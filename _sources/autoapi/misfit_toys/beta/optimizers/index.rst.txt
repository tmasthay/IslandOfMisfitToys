.. py:function:: get_optimizer(*, name, **kw)

   Get an optimizer based on the specified name.

   :param name: The name of the optimizer. Supported values are 'adam', 'sgd', and 'lbfgs'.
   :type name: str
   :param \*\*kw: Additional keyword arguments to be passed to the optimizer constructor.

   :returns: A list containing the optimizer class and the keyword arguments.
   :rtype: list

   :raises ValueError: If the specified optimizer name is unknown.
